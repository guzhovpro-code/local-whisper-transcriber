# Local Whisper Transcriber

Локальный голосовой набор для Windows. Нажал кнопку — говоришь — получаешь текст. Распознавание происходит **полностью на вашем компьютере**: ничего не уходит в облако, работает без интернета, без подписок и без лимитов.

- Модель: Whisper large-v3-turbo (faster-whisper), русский + английские термины латиницей
- История: каждая диктовка автоматически сохраняется — ничего не теряется, даже если нажали Clear
- Фильтр знаменитых галлюцинаций Whisper («Субтитры делал DimaTorzok», «Продолжение следует» и др.)
- Видеокарта NVIDIA не обязательна: без неё работает на процессоре (медленнее)

## Установка

Нужен [Python 3.10+](https://www.python.org/downloads/) (при установке отметьте галочку «Add Python to PATH»).

```
pip install -r requirements.txt
python transcriber.py
```

При первом запуске модель (~1.6 ГБ) скачается автоматически с Hugging Face — это разовая операция, дальше всё работает офлайн.

Запуск без чёрного окна консоли: `pythonw transcriber.py` (можно сделать ярлык).

## Как пользоваться

1. Выберите микрофон, нажмите **RECORD**, говорите, нажмите **STOP** — текст появится в окне.
2. **Copy to clipboard** — забрать текст, **Clear** — очистить окно.
3. Вкладка **История** — все диктовки с датой и временем: клик по записи → полный текст → Copy. Кнопка «Очистить историю…» убирает старые записи в архивный файл (безвозвратно ничего не удаляется).

История хранится в `~/transcriber-history.jsonl`, архив — в `~/transcriber-history-archive.jsonl`.

## Требования

- Windows 10/11, Python 3.10+
- ~2 ГБ на диске под модель
- С видеокартой NVIDIA (от 4 ГБ VRAM) — распознавание за секунды; без неё — на CPU, медленнее
